
==========================================
 NULL SERVER:

To run the null/mock server in a new terminal:

Python3:
   1) Open a new terminal window
   2) Change directory to the nullserver project: {{PY_DIR}}
   3) Start the null server using the port from the env file:
       {{START_CMD_PY}}

.NET:
   1) Open a new terminal window
   2) Change directory to the .NET nullserver project: {{DOTNET_DIR}}
   3) Start the .NET null server using the port from the env file:
       {{START_CMD_DOTNET}}


- The .NET nullserver reads 'port' from appsettings.json; using --urls overrides the listening address.
